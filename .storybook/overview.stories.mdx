# SemLookP Widgets

<div style={{ display: 'flex', justifyContent: 'space-between', alignItems: 'center', flexWrap: 'wrap' }}>
    <div style={{ flex: '1 1 500px' }}>

## How to use the widgets

### Installation

<p style={{fontSize:18}}>This package is published as source code and compiled as a [npm package](https://github.com/nfdi4health/semlookp-widgets/releases) on [GitHub.com](https://github.com/nfdi4health/semlookp-widgets/). To install the package, you need to authenticate to GitHub.com. Detailed documentation on this topic can be found [here](https://docs.github.com/en/packages/working-with-a-github-packages-registry/working-with-the-npm-registry#authenticating-with-a-personal-access-token).</p>


<ol>
    <li style={{fontSize:18}}>Create a Personal Access Token (PAT), see [Creating a personal access token (classic)](https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/managing-your-personal-access-tokens#creating-a-personal-access-token-classic).</li>
    <li style={{fontSize:18}}>Create a local `~/.npmrc` file add the following two lines to it. Replace `TOKEN` with your personal access token</li>
```
//npm.pkg.github.com/:_authToken=TOKEN
@nfdi4health:registry=https://npm.pkg.github.com
```
<br></br>
    <li style={{fontSize:18}}>Install the package (For dependency issues, try installing with the --legacy-peer-deps flag):</li>
```
npm install @nfdi4health/semlookp-widgets
```
<p style={{fontSize:18}}>For a list of available versions, see the [release page](https://github.com/nfdi4health/semlookp-widgets/releases) or the [GitHub NPM registry](https://github.com/nfdi4health/semlookp-widgets/pkgs/npm/semlookp-widgets)</p>
<br></br>
    <li style={{fontSize:18}}>Install peer dependencies (For dependency issues, try installing with the --legacy-peer-deps flag):</li>
```
npm install @elastic/eui@62.1.0 @emotion/react@11.11.3 react@17.0.2 react-dom@17.0.2 react-query@3.39.2
```
<br></br>
    </ol>

### Implementation in a React app

<p style={{fontSize:18}}>A demo project can be found [here](https://github.com/nfdi4health/semlookp-widgets-demo).</p>

<ol>
    <li style={{fontSize:18}}>In the 'Docs' tab of the widgets in this Storybook, a code snippet is provided in the `Show code` drop-down below the sample widget. Copy the snippet, add into your app and replace with your parameters.</li>
    <li style={{fontSize:18}}>Wrap components with the Elastic UI Provider (required for the ElasticUI design library)</li>
```
import {EuiProvider} from "@elastic/eui";
<EuiProvider>YOUR COMPONENTS</EuiProvider>
```
<br></br>
    <li style={{fontSize:18}}>Wrap the components in a QueryClientProvider (required for the react-query library)</li>
```
import {QueryClient, QueryClientProvider} from "react-query";
const queryClient = new QueryClient();
<QueryClientProvider client={queryClient}>YOUR COMPONENTS</QueryClientProvider>
```
<br></br>
    <li style={{fontSize:18}}>Import light theme (styling)</li>
```
import "@elastic/eui/dist/eui_theme_light.css";
```
    </ol>
